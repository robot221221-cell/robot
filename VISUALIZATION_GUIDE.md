# 碰撞体与末端执行器可视化说明

## 修改总结

### ✅ 已完成的修改

#### 1. UR5e 碰撞体可视化 ✓
**问题**: UR5e的碰撞体不是红色，不方便观察  
**解决**: 在 `<default class="ur5e">` 中修改了 `collision` 类的默认配置：

```xml
<default class="collision">
  <geom type="capsule" group="3" rgba="1 0 0 0.2"/>  <!-- 半透明红色 -->
  <default class="eef_collision">
    <geom type="cylinder" rgba="1 0 0 0.2"/>  <!-- 末端也是红色 -->
  </default>
</default>
```

**结果**: UR5e现在有 **6个红色半透明碰撞体**：
- 5个胶囊体 (CAPSULE)
- 1个圆柱体 (CYLINDER) - 末端法兰

#### 2. 末端执行器可视化 ✓
**问题**: 在仿真环境中看不到末端执行器（site太小且group设置不可见）

**原因分析**:
- 原始site配置: `size="0.01"` (1cm半径，太小)
- 原始site配置: `group="4"` (默认不显示)

**解决**: 修改了site的默认配置和具体站点设置：

```xml
<!-- UR5e类默认site配置 -->
<site size="0.02" rgba="1 0 0 1" group="0"/>  <!-- 红色，2cm半径，始终可见 -->

<!-- FR3类默认site配置 -->
<site size="0.02" rgba="0 1 0 1" group="0"/>  <!-- 绿色，2cm半径，始终可见 -->

<!-- UR5e末端执行器站点 -->
<site name="ur5e_ee_site" pos="0 0.1 0" size="0.025" rgba="1 0 0 1" group="0"/>

<!-- FR3末端执行器站点 -->
<site name="fr3_ee_site" pos="0 0 0.107" quat="..." size="0.025" rgba="0 1 0 1" group="0"/>
```

**结果**: 
- **UR5e末端**: 法兰前方 **10cm** 处显示 **红色球体** (半径2.5cm)
- **FR3末端**: 法兰前方 **10.7cm** 处显示 **绿色球体** (半径2.5cm)

## 关于末端执行器位置的说明

### 为什么末端不在法兰面上？

这是机器人学中的标准做法：

1. **法兰面**: 机械臂最后一个关节的连接面
2. **TCP (Tool Center Point)**: 工具中心点，实际操作的位置

对于抓取任务，TCP通常设置在：
- 无工具时：法兰前方一定距离（如10cm）
- 有工具时：夹爪/吸盘的抓取中心

### 当前设置

**UR5e**: `pos="0 0.1 0"` 
- 沿Y轴正方向（法兰朝向）前进10cm
- 适合安装标准的两指夹爪

**FR3**: `pos="0 0 0.107"` 
- 沿Z轴正方向前进10.7cm  
- 加上旋转偏移 `quat="0.92388 0 0 -0.382683"`
- 符合FR3的标准TCP定义

### 如何修改TCP位置？

如果你要安装不同的末端工具，可以修改site的pos参数：

```xml
<!-- 示例：如果夹爪长15cm -->
<site name="ur5e_ee_site" pos="0 0.15 0" .../>

<!-- 示例：如果夹爪长8cm -->
<site name="ur5e_ee_site" pos="0 0.08 0" .../>
```

## 常见问题

**Q1: 为什么我看不到红色碰撞体？**
A: 按 `G` 键打开碰撞体显示，或在菜单中启用 Collision rendering

**Q2: 为什么我看不到末端的彩色球？**
A: 按 `T` 键打开站点显示，或在菜单中启用 Sites rendering

**Q3: 红色胶囊体是否影响仿真性能？**
A: 不会。这些是碰撞体本身，仅调整了颜色。渲染开销极小。

**Q4: 可以关闭碰撞体的可视化吗？**
A: 可以。将 `rgba="1 0 0 0.2"` 改为 `rgba="1 0 0 0"` (alpha=0全透明)，或者在viewer中按 `G` 键关闭。

**Q5: 为什么UR5e的末端碰撞体是圆柱而不是胶囊？**
A: 这是原始模型的设计，代表法兰盘的形状。你可以改为胶囊体如果需要。
